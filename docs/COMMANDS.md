# Pancake Commands


```
add
	-- Add new package or dependencies
install
	-- Install depending package
upgrade
	-- Upgrade all packages
clean
	-- Clean all data
cache
	-- Save fetched packages as a cache.
check
	-- Check pancake manifest is safe
help
	-- Man
init
	-- Initialize pancake project
licenses
	-- Show license
outdated
	-- Check outdated packages
version
	-- Show Pancake version
```



# Maple Commands

```
init
	-- Create new maple plugin
publish
	-- Publish your developed maple plugin
upload
	-- Upload built package
build
	-- Build maple python package
self-install
	-- Install maple package (.mpk) from local
	
=====================================================================================

install
	-- Install maple package from repository
remove
	-- Uninstall maple package
reconfigure
	-- Reconfigure and reinstall maple package
```
